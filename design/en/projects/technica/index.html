Technica

The website of the schoolcompany Technica Light & Sound had quite a lot of changes over the time, but the main idea stayed the same. This article will show some of my thoughts and techniques on these topics:

  • Landingpage
  • Events
  • Renting-System

Landingpage

The landing page should inform the user about every aspect of the company. In order to do this without any kind of distraction, I've decided a hero-image (image, which stretches across the whole monitor) is most suitable, paired with adequate text. The images slide in and out like in a gallery.

The companies capabilities are displayed in a list, with a matching and side-changing image. Directly below the header is a navigation with anchor-links giving the user the ability to directly scroll to the point he wants.
In the background are color stripes, which got rotated 12° using the CSS parameter transform: skewX(-12deg);. Some of them have a neutral color like grey or with a subtle outline. The colored stripes dynamically change the color based on the currently displayed hero image. This connects the top and bottom part of the landing page.

Business partners are displayed in a 3*x grid. The current selection of one is shown with a transition. In that transition, the element is getting slightly bigger and the other ones are blurred out.

"Veranstaltungen" - Events

Events are obviously the most important topic of an event-company, which is why this site is of high value.

In the top of the page are the current projects and events displayed. The most important ones in a 2*x grid, but every event in a simple list below. That way the focus is shifted to the most important ones.

Below is a list of past events. Again a 2*x grid, which contains blocks displaying event title, a short description, and an image. Once you click on an event, the whole text is shown and the user can look at all available images in a gallery. The gallery system is working on desktop and also on mobile, just like a native one. The used library is PhotoSwipe.

Renting-System

The renting system is by far the most complex thing on this website. It basically is like an online-shop, but for renting speakers, lights etc. The structure of this shop is split into 4 parts: recommendations, sound equipment, and accessories.

In order to display the items in the most pleasing way, there is a normal list option of the displayed items, and for the desktop a grid one as well.

After the user-selection, the selected equipment gets processed by javascript and database requests and a summary is calculated.

When the user then confirms the selection, the data is added to a different database. From there, an email is sent to configurable e-mail addresses. That way, the data gets fast and safely to the responsible persons.
