Live Lyrics

Live Lyrics is a Browser Addon developed for Google Chrome. The User can link their Spotify Account to the Addon and then get the lyrics displayed in real-time. I will cover the Addon-Development and the webdesign process in this article. link to the project

Addon-Development

I started the project by illustrating what I wanted to do. That was plain and simple: displaying lyrics in real-time of the currently playing song on Spotify. To implement these features, I used the APIs of Spotify and Genius.

I handled the authentification with Spotify with a Chrome-Extension exclusive function. chrome.identity.launchWebAuthFlow This emphasizes that all login-data is not saved to some server of mine.

The biggest problem I faced was the naming difference of songs between Spotify and Genius. For example, the song name on Spotify: Home (with Mashine Gun Kelly, X Ambessedors...). The song name on Genius: Home. To prevent false positives, I built an algorithm which cuts the name after certain characters (e.g. -,(,),/) and some other processing. The result still isn't 100% perfect, but way closer than before.

The Addon is still under development, but currently on hold. Thus it's not available in the Chrome Webstore.

Webdesign Process

The goal of the website was to create a simplistic and structured website, which makes good use of the available space. The most important step for this was to me, to find an appealing color-scheme, typography and consistency.

The website contains 3 subpages. This is enough to bring across all information, and it's easy on the user. A landing page, installation instructions, and news.

The landing page fulfills 2 tasks: to show all important information on first sight and being visually pleasent. The CTA (Call To Action) button is the normal state a ghost-button, which transforms into a filled one when the user is hovering the button.
